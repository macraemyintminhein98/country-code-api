Country Code API

Dead simple JSON API for ISO 3166-1 alpha-2 country codes and names.

Quickly integrate country data into your apps without database hassle.

How to use:

GET https://your-vercel-app-url/api/countries?key=YOUR_API_KEY

Replace YOUR_API_KEY with your purchased API key.

Example Response:

[
  { "code": "US", "name": "United States" },
  { "code": "CA", "name": "Canada" },
  // ... more countries
]
